Чтобы запустить проект:
1. Скачать embedding model intfloat/multilingual-e5-large (https://huggingface.co/intfloat/multilingual-e5-large) и поместить в директорию backend_copy/models/multilingual-e5-large
2. Скачать qwen3 0.6b (https://huggingface.co/Qwen/Qwen3-0.6B) и поместить в директорию backend_copy/models/qwen3_06b
3. Скачать reranker cross-encoder/ms-marco-MiniLM-L6-v2 (https://huggingface.co/cross-encoder/ms-marco-MiniLM-L6-v2) и поместить в директорию backend_copy/models/ms-marco-MiniLM-L6-v2
